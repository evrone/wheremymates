## Getting Started
### Prerequisites

WhereMyMates requires Rails 3.x

### Installation

1. Create and use any gemset.
2. `gem install bundler` 
3. `bundle install` 
4. `rake db:create`
5. `rake db:migrate`

### Usage

For run server execute `rails s`
